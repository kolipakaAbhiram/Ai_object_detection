# AI Object Detection 🤖🔍

A real-time, browser-based object detection system utilizing TensorFlow.js and the COCO-SSD model. This project processes video feeds client-side to identify and track everyday objects without the need for a backend server.

## Features
* Dual-Engine Mode support (YOLOv8 + TF.js)
* Adjustable Confidence and NMS IoU Thresholds
* Temporal Smoothing for stable bounding boxes
* Real-time FPS monitoring
